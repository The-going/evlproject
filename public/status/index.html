virt (QEMU)
BCM2837 (Raspberry 3 Model B)
Kirin 620 (HiKey LeMaker)


1 Means that we pass the pipeline torture test (see CONFIG_IRQ_PIPELINE_TORTURE_TEST).

2 Steely is an actively developed Xenomai Cobalt derivative we have been using as a workhorse (or guinea pig at times) for developing and improving Dovetail. Steely is nowhere near production software, we try fundamental changes there for going beyond Cobalt’s current limitations, which could not be merged into the mature Xenomai code base at this point. When this box is checked, the corresponding Dovetail port has been tested successfully with a Steely real-time core.

To build Steely, you will also need the user-space companion tools and libraries available at this URL.
