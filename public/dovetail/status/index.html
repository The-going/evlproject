1 Means that the pipeline torture tests pass (see CONFIG_IRQ_PIPELINE_TORTURE_TEST).

2 evenless (EVL for short) is a basic co-kernel with real-time capabilities used in testing and improving the Dovetail interface. When this box is checked, the corresponding Dovetail port has been tested successfully with this core.
