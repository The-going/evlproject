<sup>1</sup> Defines the stage switching behavior for out-of-band
callers:

- &#9651; if the core may switch the caller to out-of-band mode if
  running in-band at the time of the call.

- &#9661; if the core may switch the caller to in-band mode if running
  out-of-band at the time of the call.
